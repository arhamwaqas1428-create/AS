# AS Expenses

A modern, premium daily expenses tracker designed for precision and style.

## Features

- **AI-Powered Categorization**: Automatically suggests expense categories using Google Gemini via Genkit.
- **Daily Dashboard**: Real-time summary of your spending patterns.
- **Responsive Design**: Elegant user interface that works seamlessly across all devices.
- **Modern Tech Stack**: Built with Next.js 15, React 19, Tailwind CSS, and Shadcn UI.

## Getting Started

To run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
