# **App Name**: AS Expenses

## Core Features:

- Expense Tracking: Allows users to input and track daily expenses with categories, amounts, names, and descriptions.
- Expense Categorization: Categorizes expenses into predefined categories (Food, Transport, Shopping, Bills, Other) and provides summary.
- Total Spending Display: Calculates and displays total daily spending prominently.
- Data Storage: Stores expense data.
- Description Analysis: Analyzes expense descriptions with an AI tool, to provide context-aware category suggestions for future entries.
- UI Image Generation: Creates a hero page section image with a relevance to a finance website

## Style Guidelines:

- Dominant black background (#000000) for a premium, high-end feel.
- Vibrant yellow (#FFEA00) for buttons, highlights, and active states to create contrast and focus.
- Use 'Inter', a grotesque-style sans-serif font for both body text and headlines to ensure modern look and readability.
- Modern, minimalist icons for expense categories. The icon's shapes and metaphors should relate to finance and/or spending, and complement the selected typeface.
- Clean and spacious layout for clarity, focusing on visual hierarchy for intuitive use.
- Subtle transitions and animations to enhance user experience when adding expenses.
